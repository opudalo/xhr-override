XmlHttpRequest methods override example

Usage example: 
      XHROverride({
        methods: {
          send: function () {
            console.log('open:', arguments)
          },
          open: function () {
            console.log('send:', arguments)
          }
        },
        listeners: {
          readystatechange: function () {
            console.log('onreadystatechange:', this.readyState)
          }
        }
      })
